Resilient Distributed Datasets (RDDs)

At the heart of Spark Core is the concept of Resilient Distributed Datasets. RDDs are immutable, distributed collections of objects that can be processed in parallel. They provide fault tolerance through lineage tracking and enable efficient in-memory computation.

Immutable: Once created, RDDs cannot be changed
Distributed: Data is partitioned across cluster nodes
Resilient: Automatic fault recovery through lineage
In-memory Computing

Cache frequently accessed data in memory to speed up iterative algorithms by 10-100x

Task Scheduling

Intelligent task distribution and data locality optimization across cluster nodes

Fault Tolerance

Automatic recovery from node failures using RDD lineage information without data replication

Code Examples

See Apache Spark in action with real-world examples

Python - Word Count

Python 
# Load text file and count words
from pyspark import SparkContext

sc = SparkContext()
text_file = sc.textFile("hdfs://data.txt")

# Split lines into words and count
word_counts = text_file \
    .flatMap(lambda line: line.split(" ")) \
    .map(lambda word: (word, 1)) \
    .reduceByKey(lambda a, b: a + b)

# Collect results
results = word_counts.collect()
print(results)
Processes millions of words in seconds

Scala - SQL Query

Scala 
import org.apache.spark.sql.SparkSession

val spark = SparkSession.builder()
  .appName("Customer Analysis")
  .getOrCreate()

// Load JSON data
val customers = spark.read.json("customers.json")
customers.createOrReplaceTempView("customers")

// SQL query
val result = spark.sql("""
  SELECT region, AVG(purchase_amount) as avg_purchase
  FROM customers 
  WHERE age > 25 
  GROUP BY region 
  ORDER BY avg_purchase DESC
""")

result.show()

Stream Processing

Java 
// Real-time stream processing
StreamingContext ssc = new StreamingContext(conf, Durations.seconds(1));

// Create DStream from socket
JavaReceiverInputDStream lines = 
    ssc.socketTextStream("localhost", 9999);

// Process streams
JavaDStream words = lines.flatMap(
    x -> Arrays.asList(x.split(" ")).iterator()
);

JavaPairDStream pairs = words.mapToPair(
    s -> new Tuple2<>(s, 1)
);

JavaPairDStream wordCounts = 
    pairs.reduceByKey((i1, i2) -> i1 + i2);

wordCounts.print();
ssc.start();

Machine Learning

Python 
# MLlib example - Linear Regression
from pyspark.ml.regression import LinearRegression
from pyspark.ml.feature import VectorAssembler

# Load data
df = spark.read.csv("housing_data.csv", header=True, inferSchema=True)

# Prepare features
assembler = VectorAssembler(
    inputCols=["bedrooms", "bathrooms", "sqft"],
    outputCol="features"
)
feature_df = assembler.transform(df)

# Train model
lr = LinearRegression(featuresCol="features", labelCol="price")
model = lr.fit(feature_df)

# Make predictions
predictions = model.transform(feature_df)
predictions.select("features", "price", "prediction").show()
